# useNavigation

Takes an array of objects as an argument for menu items. It
returns an array of menu items. Each item has usefull props
such as such as onClick, isExpanded and so on.

## Argument

| type     | description                                                       |
| -------- | ----------------------------------------------------------------- |
| [object] | { label: string, to: string, icon: node, nestedMenuItems: array } |

## Example

```javascript
import { useNavigation } from 'q3-hooked';
import Navigation from './useNavigation';

export default (menuItems) => {
  const { navigationMenus } = useNavigation(menuItems);

  return <Navigation lists={navigationMenus} />;
};
```
